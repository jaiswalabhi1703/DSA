# DSA Command Center

An interactive, phase-by-phase DSA pattern library built for technical-interview prep.
Each topic maps every pattern to its **keyword → pseudocode → time/space complexity**.

## Live structure
- `index.html` — 3D hub / launcher (start here)
- `theme.css` — shared dark-glass theme
- `*_roadmap.html` — 14 standalone topic pages
- `dsa_app.html` — the entire app bundled into a single self-contained file

## Topics
Trees · Graphs · Dynamic Programming · Stack · Arrays & Two Pointers ·
Sliding Window · Binary Search · Linked List · Hashing · Heap / Priority Queue ·
Backtracking · Greedy · Trie · Bit Manipulation

## Run locally
Open `index.html` in any browser (keep all files in the same folder),
or just open `dsa_app.html` on its own.

## GitHub Pages
Push to a repo and enable Pages on the `main` branch (root).
`index.html` is served automatically — no build step.

Display fonts load from Google Fonts; everything else works offline.
