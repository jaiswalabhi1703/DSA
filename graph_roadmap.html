DFS + BFS template
void dfs(int[][] grid, boolean[][] vis, int r, int c) { if (r<0||r>=R||c<0||c>=C||vis[r][c]||grid[r][c]==0) return; vis[r][c] = true; for (int[] d : dirs) // {{0,1},{0,-1},{1,0},{-1,0}} dfs(grid, vis, r+d[0], c+d[1]); } // caller: for each unvisited land → dfs() → count++ // BFS multi-source (01 Matrix) Queue<int[]> q = new LinkedList<>(); for (all 0-cells) { q.offer(cell); dist[r][c]=0; } while (!q.isEmpty()) { int[] cur = q.poll(); for (int[] d : dirs) { if (inBounds && dist[nr][nc] > dist[r][c]+1) { dist[nr][nc] = dist[r][c]+1; q.offer(new int[]{nr,nc}); } } }
Pattern: Mark visited BEFORE recursing (DFS) or BEFORE enqueue (BFS). Multi-source BFS = seed all sources at dist=0 simultaneously.

Dijkstra template
int[] dist = new int[V]; Arrays.fill(dist, INF); dist[src] = 0; PriorityQueue<int[]> pq = new PriorityQueue<>((a,b)->a[0]-b[0]); // {dist,node} pq.offer(new int[]{0, src}); while (!pq.isEmpty()) { int[] cur = pq.poll(); int d=cur[0], u=cur[1]; if (d > dist[u]) continue; // stale — skip for (int[] e : adj.get(u)) { // e={v,weight} if (dist[u]+e[1] < dist[e[0]]) { dist[e[0]] = dist[u]+e[1]; pq.offer(new int[]{dist[e[0]], e[0]}); } } } // Network Delay: return max(dist[]) // K-stops: use dist[k][node] 2D, BFS level-capped
Key: Never skip the stale-entry check if (d > dist[u]) continue — causes TLE without it.

Kahn's topo sort + cycle check
int[] indegree = new int[V]; for (int u=0; u<V; u++) for (int v : adj.get(u)) indegree[v]++; Queue<Integer> q = new LinkedList<>(); for (int i=0; i<V; i++) if (indegree[i]==0) q.offer(i); List<Integer> order = new ArrayList<>(); while (!q.isEmpty()) { int u = q.poll(); order.add(u); for (int v : adj.get(u)) if (--indegree[v]==0) q.offer(v); } // cycle exists if order.size() < V // DFS cycle (directed): 3-color // 0=unvisited 1=in-stack 2=done if (color[v]==1) return true; // back edge = cycle // Undirected: vis[nei] && nei!=parent → cycle
Pattern: Kahn's = indegree[] + BFS. If order.size() < V → cycle. DFS topo = postorder reversed.

PHASE 4 · UNION-FIND (DISJOINT SET UNION)
Dynamic connectivity · O(α(n)) per op
Questions
1.UFNumber of Provinces
2.UFRedundant Connection
3.UFAccounts Merge
4.UFMaking Large Island

Redundant connection = first edge where find(u)==find(v) before union.
Provinces = count nodes where find(i)==i.
Union-Find with path compression + rank
int[] parent, rank; void init(int n) { parent = new int[n]; rank = new int[n]; for (int i=0; i<n; i++) parent[i]=i; } int find(int x) { // path compression if (parent[x] != x) parent[x] = find(parent[x]); return parent[x]; } boolean union(int x, int y) { int px=find(x), py=find(y); if (px==py) return false; // already connected if (rank[px]<rank[py]) { int t=px; px=py; py=t; } parent[py]=px; if (rank[px]==rank[py]) rank[px]++; return true; }
Pattern: Path compression + union by rank → near O(1). Use HashMap<String,String> parent for string-keyed problems like Accounts Merge.

PHASE 5 · SCC, BRIDGES & ARTICULATION POINTS
Tarjan's · disc[] + low[]
Questions
1.DFSCritical Connections (Bridges)
2.DFSArticulation Points
3.DFSStrongly Connected Components
4.DFSNumber of SCCs (Kosaraju)

Bridge: low[v] > disc[u]
AP: low[v] >= disc[u] + root child check
Kosaraju: DFS order → reverse edges → DFS again
Tarjan's bridge-finding
int timer = 0; int[] disc, low; boolean[] vis; void dfs(int u, int par) { vis[u] = true; disc[u] = low[u] = timer++; for (int v : adj.get(u)) { if (!vis[v]) { dfs(v, u); low[u] = Math.min(low[u], low[v]); if (low[v] > disc[u]) // bridge! bridges.add(List.of(u, v)); } else if (v != par) low[u] = Math.min(low[u], disc[v]); } } // Kosaraju SCC: // 1) DFS on G → push to stack on finish // 2) Build reverse graph G^T // 3) DFS on G^T in stack pop order → each DFS = 1 SCC
Pattern: disc[] = discovery time, low[] = lowest reachable. Bridge: low[v]>disc[u]. AP: low[v]>=disc[u] (non-root node).

Memoized DFS + Bitmask BFS
int[][] memo; // LIP: memo[i][j] = best from (i,j) int dfs(int[][] g, int r, int c) { if (memo[r][c] != 0) return memo[r][c]; int best = 1; for (int[] d : dirs) { int nr=r+d[0], nc=c+d[1]; if (inBounds(nr,nc) && g[nr][nc]>g[r][c]) best = Math.max(best, 1+dfs(g,nr,nc)); } return memo[r][c] = best; } // Bitmask BFS — visit all nodes // state = {node, visitedMask, dist} int FULL = (1<<n)-1; Queue<int[]> q = new LinkedList<>(); for (int i=0; i<n; i++) q.offer(new int[]{i,1<<i,0}); while (!q.isEmpty()) { int[] cur = q.poll(); if (cur[1]==FULL) return cur[2]; for (int nei : adj.get(cur[0])) q.offer(new int[]{nei, cur[1]|(1<<nei), cur[2]+1}); }
Pattern: Grid = DAG → memoize DFS. Bitmask BFS: state=(node,mask). Use visited set on (node,mask) pairs. Works when nodes ≤ 20.

PHASE 7 · MINIMUM SPANNING TREE
Kruskal (sort+UF) · Prim (greedy heap)
Questions
1.KruskalMin Cost Connect All Points
2.PrimClassic MST
3.KruskalOptimize Water Distribution

Kruskal: sort edges + UF. Better sparse graphs.
Prim: Dijkstra-like heap. Better dense graphs.
Both yield same MST total weight.
Kruskal + Prim
// Kruskal Collections.sort(edges, (a,b)->a[2]-b[2]); init(V); int mstCost=0, used=0; for (int[] e : edges) { // {u, v, weight} if (union(e[0],e[1])) { mstCost += e[2]; if (++used==V-1) break; } } // Prim boolean[] inMST = new boolean[V]; PriorityQueue<int[]> pq = new PriorityQueue<>((a,b)->a[0]-b[0]); // {cost,node} pq.offer(new int[]{0,0}); while (!pq.isEmpty()) { int[] cur=pq.poll(); int u=cur[1]; if (inMST[u]) continue; inMST[u]=true; mstCost+=cur[0]; for (int[] e : adj.get(u)) if (!inMST[e[0]]) pq.offer(new int[]{e[1],e[0]}); }
Pattern: Kruskal = sort + union(). Stop at V-1 edges. Water Distribution trick: add virtual node 0 with well-cost edges.
