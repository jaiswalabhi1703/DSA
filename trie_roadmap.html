TRIE

Prefix trees, autocomplete, wildcard
CORE IDEA: A prefix tree where each node has up to 26 children. Gives O(L) insert/search by word length L, independent of how many words are stored. Ideal for prefix queries and autocomplete.
1
Trie Construction & Search
Time O(L)Space O(26·N·L)
Questions
1.Implement Trie (Prefix Tree)
2.Design Add & Search Words (wildcard)
3.Word Search II
4.Replace Words
5.Maximum XOR of Two Numbers (bit-trie)
Keyword: "prefix", "starts with", "dictionary of words", "autocomplete", "search with wildcard". L = word length.
Pseudocode
class TrieNode { TrieNode[] children = new TrieNode[26]; boolean isEnd; } void insert(String word) { TrieNode node = root; for (char c : word.toCharArray()) { int i = c - 'a'; if (node.children[i]==null) node.children[i] = new TrieNode(); node = node.children[i]; } node.isEnd = true; } boolean search(String word) { TrieNode node = root; for (char c : word.toCharArray()) { node = node.children[c-'a']; if (node==null) return false; } return node.isEnd; } // Wildcard '.': recurse over all 26 children
Pattern: Walk character by character. isEnd marks complete words. For wildcards, branch into all children. Combine with backtracking for Word Search II.

Pseudocode
// Word Search II shape void dfs(int r, int c, TrieNode node) { char ch = board[r][c]; TrieNode next = node.children[ch-'a']; if (next == null) return; if (next.word != null) ans.add(next.word); board[r][c] = '#'; for (int[] d : dirs) if (inside(r+d[0], c+d[1])) dfs(r+d[0], c+d[1], next); board[r][c] = ch; }
Pattern: DFS walks the board and the trie at the same time. Stop immediately when the current prefix is absent.

Pseudocode
// Bit trie max XOR query int bestXor(int x) { TrieNode node = root; int ans = 0; for (int bit=31; bit>=0; bit--) { int cur = (x >> bit) & 1; int want = 1 - cur; if (node.child[want] != null) { ans |= (1 << bit); node = node.child[want]; } else { node = node.child[cur]; } } return ans; }
Pattern: Character trie optimizes prefixes; bit trie optimizes bit choices from high to low.
