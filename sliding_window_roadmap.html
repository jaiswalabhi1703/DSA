SLIDING WINDOW

Fixed and variable-size windows
CORE IDEA: Maintain a contiguous window [left, right]. Expand right to include, shrink left to satisfy a constraint. Turns O(n²) substring/subarray problems into O(n).
1
Fixed-Size Window
Time O(n)Space O(1)
Questions
1.Maximum Average Subarray I
2.Find All Anagrams in a String
3.Permutation in String
4.Sliding Window Maximum (deque)
Keyword: "subarray/substring of size k", "fixed length", "all windows of length k". Window size never changes — slide it.
Pseudocode
int windowSum = 0; for (int i=0; i<k; i++) windowSum += nums[i]; // first int best = windowSum; for (int i=k; i<n; i++) { windowSum += nums[i] - nums[i-k]; // add new, drop old best = Math.max(best, windowSum); } // Anagram: keep a 26-char count array, compare each slide
Pattern: Add incoming element, remove outgoing element (i-k). Constant-time window update.
2
Variable-Size Window (longest/shortest)
Time O(n)Space O(k)
Questions
1.Longest Substring Without Repeating Chars
2.Minimum Window Substring
3.Longest Repeating Character Replacement
4.Fruit Into Baskets
5.Minimum Size Subarray Sum
Keyword: "longest/shortest substring", "at most K distinct", "smallest subarray ≥ target". Window grows and shrinks dynamically.
Pseudocode
int left = 0, best = 0; Map<Character,Integer> count = new HashMap<>(); for (int right=0; right<n; right++) { count.merge(s.charAt(right), 1, Integer::sum); // expand while (/* window invalid */ count.size() > k) { char lc = s.charAt(left); // shrink count.merge(lc, -1, Integer::sum); if (count.get(lc)==0) count.remove(lc); left++; } best = Math.max(best, right - left + 1); // record }
Pattern: for-loop expands right; inner while shrinks left until valid. For "longest" → record after shrink; for "shortest" → record inside shrink loop.
Window typeTimeSpace
Fixed-sizeO(n)O(1) or O(26)
Variable-size + HashMapO(n)O(k) distinct
