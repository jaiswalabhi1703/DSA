ARRAYS & TWO POINTERS

Opposite-ends, fast/slow, prefix sums
CORE IDEA: Two Pointers reduces O(n²) brute force to O(n) by moving two indices intelligently. Sorted input is the biggest hint. Variants: opposite-ends, same-direction (fast/slow), and partition.
1
Opposite-Ends Two Pointers
Time O(n)Space O(1)
Questions
1.Two Sum II (sorted)
2.3Sum / 4Sum
3.Container With Most Water
4.Valid Palindrome
5.Trapping Rain Water (2-ptr)
Keyword: "sorted array", "pair that sums to target", "max area between two lines", "palindrome check". Two indices move toward each other.
Pseudocode
int l=0, r=n-1; while (l < r) { int sum = nums[l] + nums[r]; if (sum == target) return new int[]{l, r}; else if (sum < target) l++; // need bigger else r--; // need smaller } // 3Sum: sort, fix i, two-pointer on (i+1, n-1) // skip duplicates: while(l<r && nums[l]==nums[l-1]) l++;
Pattern: Sort first (if order doesn't matter). Shrink window from the end that can't improve the answer.
2
Fast & Slow Pointers
Time O(n)Space O(1)
Questions
1.Remove Duplicates from Sorted Array
2.Move Zeroes
3.Linked List Cycle (Floyd's)
4.Find the Duplicate Number
5.Sort Colors (Dutch flag, 3-ptr)
Keyword: "in-place", "remove/move elements", "detect cycle", "find duplicate without extra space". One slow writer, one fast reader.
Pseudocode
// Move non-zeros / dedupe in place int slow = 0; for (int fast=0; fast<n; fast++) { if (nums[fast] != 0) { // condition to keep swap(nums, slow, fast); slow++; } } // Dutch National Flag (Sort Colors 0,1,2) int lo=0, mid=0, hi=n-1; while (mid <= hi) { if (nums[mid]==0) swap(nums, lo++, mid++); else if (nums[mid]==1) mid++; else swap(nums, mid, hi--); }
Pattern: Slow pointer marks the boundary of "processed/valid" region; fast pointer scans ahead.
3
Prefix Sum / Difference Array
Time O(n)Space O(n)
Questions
1.Subarray Sum Equals K
2.Range Sum Query - Immutable
3.Product of Array Except Self
4.Continuous Subarray Sum
5.Car Pooling (difference array)
Keyword: "sum of subarray", "range query", "count subarrays with sum K", "range update". Combine prefix sum + HashMap for counting.
Pseudocode
// Subarray Sum = K (prefix sum + hashmap) Map<Integer,Integer> freq = new HashMap<>(); freq.put(0, 1); // empty prefix int sum=0, count=0; for (int num : nums) { sum += num; count += freq.getOrDefault(sum - k, 0); freq.merge(sum, 1, Integer::sum); } // O(n) time, O(n) space — beats O(n²) brute force
Pattern: prefix[i] = sum of first i elements. Range sum = prefix[r+1] - prefix[l]. For counting, store prefix frequencies in a map.
4
Kadane / Running Best
Time O(n)Space O(1)
Questions
1.Maximum Subarray
2.Maximum Product Subarray
3.Best Time to Buy and Sell Stock
4.Maximum Sum Circular Subarray
5.Maximum Alternating Subsequence Sum
6.Product of Array Except Self
Keyword: "maximum subarray/product/profit", "best ending here", "running min/max". Keep the best local state and global answer.
Pseudocode
// Kadane: max subarray sum int cur = nums[0], best = nums[0]; for (int i=1; i<n; i++) { cur = Math.max(nums[i], cur + nums[i]); best = Math.max(best, cur); } // Max product: track both max and min ending here // because negative * negative can become max
Pattern: Decide whether the best segment ending at i should extend the previous segment or restart at i.
5
Index Marking, Cyclic Sort & Matrix Arrays
Time O(n)Space O(1)
Questions
1.First Missing Positive
2.Find All Numbers Disappeared in an Array
3.Find the Duplicate Number
4.Set Matrix Zeroes
5.Rotate Image
6.Spiral Matrix
7.Game of Life
Keyword: "numbers are 1..n", "in-place O(1)", "matrix transform". Use indices as storage, or layer-by-layer matrix traversal.
Pseudocode
// First Missing Positive: place x at index x-1 for (int i=0; i<n; i++) { while (nums[i] >= 1 && nums[i] <= n && nums[nums[i]-1] != nums[i]) { swap(nums, i, nums[i]-1); } } for (int i=0; i<n; i++) if (nums[i] != i+1) return i+1; return n+1; // Matrix rotate: transpose, then reverse each row
Pattern: When values map to indices, the array can become its own hash table. For matrices, track boundaries or transform in layers.
