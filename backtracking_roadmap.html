BACKTRACKING

Subsets, permutations, grid solving
CORE IDEA: Build candidates incrementally; abandon ("backtrack") a path the moment it can't lead to a solution. Choose → explore → un-choose. Exponential, but pruning makes it tractable.
1
Subsets / Combinations / Permutations
Time O(2ⁿ)/O(n!)Space O(n)
Questions
1.Subsets I & II
2.Combination Sum I & II
3.Permutations I & II
4.Letter Combinations of a Phone Number
5.Palindrome Partitioning
Keyword: "all combinations", "all subsets", "all permutations", "generate every...". The branching factor and the "start index" distinguish the three families.
Pseudocode
// Generic backtracking skeleton void backtrack(List<Integer> path, int start) { result.add(new ArrayList<>(path)); // record (subsets) for (int i=start; i<n; i++) { // prune: skip duplicates / invalid choices if (i>start && nums[i]==nums[i-1]) continue; path.add(nums[i]); // CHOOSE backtrack(path, i+1); // EXPLORE path.remove(path.size()-1); // UN-CHOOSE } } // Permutations: use boolean[] used instead of start index // Combination Sum (reuse): recurse with i, not i+1
Pattern: Subsets/combinations use a start index (forward-only). Permutations use a used[] flag (any unused element). Sort + skip nums[i]==nums[i-1] dedupes.
2
Grid & Constraint Backtracking
Time exp.Space O(n²)
Questions
1.N-Queens
2.Sudoku Solver
3.Word Search
4.Word Search II (with Trie)
Keyword: "place N items with constraints", "solve the board", "find word in grid". Mark visited, recurse, unmark.
Pseudocode
// Word Search — DFS with visited marking boolean dfs(int r, int c, int idx) { if (idx==word.length()) return true; if (outOfBounds || board[r][c]!=word.charAt(idx)) return false; char tmp = board[r][c]; board[r][c] = '#'; // mark visited boolean found = dfs(r+1,c,idx+1) || dfs(r-1,c,idx+1) || dfs(r,c+1,idx+1) || dfs(r,c-1,idx+1); board[r][c] = tmp; // restore (backtrack) return found; }
Pattern: Modify state to mark a choice, recurse, then restore it. N-Queens prunes with column/diagonal sets for O(1) validity checks.
3
Partitioning & String Backtracking
Time exp.Space O(n)
Questions
1.Restore IP Addresses
2.Palindrome Partitioning
3.Word Break II
4.Expression Add Operators
5.Generate Parentheses
6.Different Ways to Add Parentheses
Keyword: "split string into valid pieces", "all valid expressions", "all sentences". The choice is usually the next cut position.
Pseudocode
// Partition string by choosing the next cut void dfs(int idx, List<String> path) { if (idx == s.length()) { ans.add(new ArrayList<>(path)); return; } for (int end=idx; end<s.length(); end++) { if (!isValid(s, idx, end)) continue; path.add(s.substring(idx, end+1)); dfs(end+1, path); path.remove(path.size()-1); } }
Pattern: At index i, try every valid next segment. Precompute validity when checks are repeated, like palindrome ranges.
4
Pruning, Branch & Bound
Time exp.Space O(n)
Questions
1.Combination Sum III
2.Partition to K Equal Sum Subsets
3.Matchsticks to Square
4.Beautiful Arrangement
5.Optimal Account Balancing
6.Shopping Offers
Keyword: "partition into k groups", "minimize choices", "constraints are tight". Sort, reject impossible states early, and memoize state when possible.
Pseudocode
// K equal subsets: fill buckets with pruning boolean dfs(int idx) { if (idx == nums.length) return true; for (int b=0; b<k; b++) { if (bucket[b] + nums[idx] > target) continue; bucket[b] += nums[idx]; if (dfs(idx+1)) return true; bucket[b] -= nums[idx]; if (bucket[b] == 0) break; // avoid symmetric empty buckets } return false; }
Pattern: Sort hard choices first. Prune overflow, duplicate choices, symmetric buckets, and states that cannot beat the current best.
